Fall Detection using CNN and RNN ðŸ‘´

Using sensor data, detect if the user is falling or not (2017-18)

In this repository, I have implemented two fall detection algorithms based on sensor data (accelerometer and gyroscope) to classify the movements of a person as either `fall` or `non-fall`. The implementation is done using Tensorflow's `layers` API. The implementation uses a Convolutional Neural Network / Recurrent Neural Network followed by several dense layers. The final output is a single sigmoid score (1 = fall, 0 = non-fall). A Mean squared error metric is used as the loss function and optimized using AdamOptimizer.

(2017-18)

Project Links:

In my final year's capstone project, we created a functinoal semi-autonomous self driving car prototype. We built it from scratch including the car and its chassis as well as the end-to-end Convolutional Neural Network model to turn RGB camera input to steering angle and speed. We also introduced an android app in the middle to convert any voice commands from user to override the default behavior (e.g. user can say 'turn left' which will be read by the system and the car will take left turn on next possible left turn).

The model generalized well on unique track scenarios. A big shout out to my teammates Nikhil Sharma, Gaurav Arora, Harjot Singh, Prince Dhaliwal for their excellent contributions to this project.

Tensorflow | Data Augmentation | Convolutional Neural Networks | Speech-to-text (Android built-in) | Arduino | Circuit and power design

P.S. this is not a line-follower. The car decides steering angle and speed based on RGB input (using neural network model).
